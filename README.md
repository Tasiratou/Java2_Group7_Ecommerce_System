**Spring Boot E-Commerce REST API**

# Project Overview
This project is a Spring Boot RESTful API developed by group 7 members. It provides basic e-commerce product management functionalities using CRUD (Create, Read, Update and Delete) operations.

The project follows a layered architecture consisting of:
- Controller Layer
- Service Layer
- Repository Layer
- Model (Entity) Layer
- Exception Handling

 # Technologies Used
- Java 17 (or your version)
- Spring Boot
- Spring Data JPA
- Maven
- Lombok
- PostgreSQL(pgAdmin) Database
- Postman
- Git & GitHub

# API Endpoints
| Method | Endpoint | Description |
|---------|----------|-------------|
| POST | /api/products | Create Product |
| GET | /api/products | Get All Products |
| GET | /api/products/{id} | Get Product by ID |
| PUT | /api/products/{id} | Update Product |
| DELETE | /api/products/{id} | Delete Product |

# Database
The application uses Spring Data JPA for database interaction.
